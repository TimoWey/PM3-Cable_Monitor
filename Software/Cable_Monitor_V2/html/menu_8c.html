Detailed Description

The menu of the graphical user interface(GUI)

Initializes and displays the menu.

Note
Provides the function MENU_check_transition() for polling user actions.
The variable MENU_transition is set to the touched menu item.
If no touch has occurred the variable MENU_transition is set to MENU_NONE
The different steps of the menu are executed in MENU_main().
The function MENU_get_transition() returns the new menu item.

static

Check for selection/transition.

Parameters
[in]typeActual menu type

If the last transition has been consumed (MENU_NONE == MENU_transition)
and the touchscreen has been touched for a defined period
the variable MENU_transition is set to the touched item.

Note
Evalboard revision E (blue PCB) has an inverted y-axis in the touch controller compared to the display.
Uncomment or comment the #define EVAL_REV_E in main.h accordingly.
