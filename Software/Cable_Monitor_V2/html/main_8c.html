Detailed Description

Sets up the microcontroller, the clock system and the peripherals.

Initialization is done for the system, the user LEDs, and the LCD display with the touchscreen.
Then the code enters an infinite while-loop in which it calls the menu.

static void MAIN_gyro_disable ( void  )
static

Disable the GYRO on the microcontroller board.

Disable GYRO on PC1 and PF8 and set them as an analog input.

Note
MISO of the GYRO is connected to PF8 and CS to PC1.
Some times the GYRO goes into an undefined mode at startup and pulls the MISO low or high thus blocking the analog input on PF8.
The simplest solution is to pull the CS of the GYRO low for a short while which is done with the code below.
PF8 is also reconfigured.
An other solution would be to remove the GYRO from the microcontroller board by unsoldering it.
