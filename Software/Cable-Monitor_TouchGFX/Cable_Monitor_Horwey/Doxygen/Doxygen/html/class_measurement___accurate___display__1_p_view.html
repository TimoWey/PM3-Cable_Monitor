◆ accu_meas()

void accu_meas ( )
virtual

Performs accurate measurement and updates the display accordingly.

This function calculates the accurate measurement values for a single phase and updates the display with the calculated values. If the standard deviation is too high, an error message is displayed. If there are errors with the current, frequency, or distance values, appropriate error messages are displayed.

Note
This function disables TouchGFX interrupts during the calculation of accurate measurement values.
This function assumes that the ACCU_MEAS struct is defined and the accurate_measurement() function is implemented.
This function is only executed if the SIMULATOR macro is not defined.
