Cable-Monitor-Code

This code_cable_monitor is the firmware for the Electronics-Project Modules ET.PM3: Cable-Monitor
A cable monitor had to be developed. It should sense a mains cable via the electrostatic, as well as electromagnetic field. Furthermore, it should be able to measure the distance and the current flowing through it. The values are then displayed on a touch display.


The file measuring.c prepares the measured data.
calculation.c performs the necessary calculations for displaying the measurements.
While the file menu.c takes care of the menu change and the touchscreen.
display.c determines the visuals of the display.
main.c puts everything together.

This code is written for the evaluation board 32F429IDISCOVERY (Discovery kit with STM32F429ZI MCU).

How to use

With this firmware you are able to perform distance, angle and current measurements of mains cables. There is also the possibility for a simple and fast calibration.
The use of the menu should be self-explanatory. Everything is described on the respective page.
