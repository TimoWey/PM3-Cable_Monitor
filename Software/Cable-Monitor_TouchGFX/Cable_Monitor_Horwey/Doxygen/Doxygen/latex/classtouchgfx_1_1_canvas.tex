\doxysubsection{Detailed Description}
Class for easy rendering using \doxylink{classtouchgfx_1_1_canvas_widget_renderer}{Canvas\+Widget\+Renderer}.

The \doxylink{classtouchgfx_1_1_canvas}{Canvas} class will make implementation of a new \doxylink{classtouchgfx_1_1_canvas_widget}{Canvas\+Widget} very easy. The few simple primitives allows moving a "{}pen"{} and drawing the outline of a shape which can then be rendered.

The \doxylink{classtouchgfx_1_1_canvas}{Canvas} class has been optimized to eliminate drawing unnecessary lines outside the currently invalidated rectangle.
