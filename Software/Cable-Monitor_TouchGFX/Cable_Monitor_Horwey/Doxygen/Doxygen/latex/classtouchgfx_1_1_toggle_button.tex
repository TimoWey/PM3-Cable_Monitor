\doxysubsection{Detailed Description}
A \doxylink{classtouchgfx_1_1_toggle_button}{Toggle\+Button} is a \doxylink{classtouchgfx_1_1_button}{Button} specialization that swaps the two bitmaps when clicked, such that the previous "{}pressed"{} bitmap, now becomes the one displayed when button is not pressed. This can by used to give the effect of a button that can be pressed in and when it is subsequently pressed, it will pop back out.

Allows the \doxylink{classtouchgfx_1_1_toggle_button}{Toggle\+Button} to be forced into either the pressed state, or the normal state. In the pressed state, the \doxylink{classtouchgfx_1_1_button}{Button} will always be shown as pressed down (and shown as released when the user presses it). In the normal state, the \doxylink{classtouchgfx_1_1_button}{Button} will be show as released or pressed depending on its actual state.

The images are expected to be of the same dimensions, and the \doxylink{classtouchgfx_1_1_button}{Button} is resized to the dimensions of the pressed \doxylink{classtouchgfx_1_1_bitmap}{Bitmap}.

\begin{DoxyNote}{Note}
