Support formats\+: \%c (element type\+: char), \%s (element type\+: null-\/terminated Unicode\+Char list), \%u, \%i, \%d, \%o, \%x (all these are integers formatted in radix 10, 10, 10, 8, 16 respectively).

The number formats (\%u, \%i, \%d, \%o and \%x) all support \begin{DoxyVerb}\%[flags][width][.precision]X
\end{DoxyVerb}
 Where flags can be\+:
\begin{DoxyItemize}

\item \textquotesingle{}+\textquotesingle{}\+: force sign.
\item \textquotesingle{} \textquotesingle{}\+: insert space if value is positive.
\item \textquotesingle{}0\textquotesingle{}\+: left pad with zeros instead of spaces (see width).
\end{DoxyItemize}
