\doxysubsection{Detailed Description}
\subsubsection*{template$<$class T$>$\newline
class touchgfx\+::\+Icon\+Button\+Style$<$ T $>$}
An icon button style. This class is supposed to be used with one of the Button\+Trigger classes to create a functional button. This class will show one of two icons depending on the state of the button (pressed or released).

To get a background behind the icon, use \doxylink{classtouchgfx_1_1_icon_button_style}{Icon\+Button\+Style} together with e.\+g. \doxylink{classtouchgfx_1_1_image_button_style}{Image\+Button\+Style}\+: \doxylink{classtouchgfx_1_1_icon_button_style}{Icon\+Button\+Style}\texorpdfstring{$<$}{<}Image\+Button\+Style$<$\+Click\+Button\+Trigger$>$ \texorpdfstring{$>$}{>} my\+Button;

The \doxylink{classtouchgfx_1_1_icon_button_style}{Icon\+Button\+Style} will center the icon on the enclosing container (normally \doxylink{classtouchgfx_1_1_abstract_button_container}{Abstract\+Button\+Container}). Set the size of the button before setting the icons.

The position of the icon can be adjusted with set\+Icon\+XY.
