\doxysubsection{Detailed Description}
Abstract interface for classes implementing general 2D vector drawing.

Users of this interface are Widgets drawing vector graphics like SVG images.

Implementations draw the path given (in draw\+Path) using the last color or gradient supplied with set\+Color or set\+Linear\+Gradient.
