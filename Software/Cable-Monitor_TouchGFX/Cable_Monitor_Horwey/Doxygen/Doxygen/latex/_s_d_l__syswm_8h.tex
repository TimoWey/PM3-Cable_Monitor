\doxysubsection{Detailed Description}
Include file for SDL custom system window manager hooks.

Your application has access to a special type of event \doxylink{_s_d_l__events_8h_a3b589e89be6b35c02e0dd34a55f3fccaa73749d735a18ce6ef17a09ee70d5dbe7}{SDL\+\_\+\+SYSWMEVENT}, which contains window-\/manager specific information and arrives whenever an unhandled window event occurs. This event is ignored by default, but you can enable it with \doxylink{_s_d_l__events_8h_afb772893e1c46f186fa39a4defe76df3}{SDL\+\_\+\+Event\+State()}.
