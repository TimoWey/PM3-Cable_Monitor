\doxysubsection{Detailed Description}
The \doxylink{classtouchgfx_1_1_video_controller}{Video\+Controller} is an abstract interface for the video decoder. A concrete implementation will be generated by the Touch\+GFX Generator.

The \doxylink{classtouchgfx_1_1_video_controller}{Video\+Controller} can control multiple video streams in multiple Video\+Widgets. These are recognized by their Handle.

\doxylink{classtouchgfx_1_1_application}{Application} code should only interact with the \doxylink{classtouchgfx_1_1_video_widget}{Video\+Widget} class.
