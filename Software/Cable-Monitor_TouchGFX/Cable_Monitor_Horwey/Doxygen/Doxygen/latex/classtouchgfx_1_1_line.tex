\doxysubsection{Detailed Description}
Simple \doxylink{classtouchgfx_1_1_canvas_widget}{Canvas\+Widget} capable of drawing a line from one point to another point. The end points can be moved to new locations and the line width can be set and changed. A 10 pixel long line along the top of the screen with a width on 1 pixel has endpoints in (0, 0.\+5) and (10, 0.\+5) and line width 1. The \doxylink{classtouchgfx_1_1_line}{Line} class calculates the corners of the shape, which in this case would be (0, 0), (10, 0), (10, 1) and (0, 1) and tells \doxylink{classtouchgfx_1_1_canvas_widget_renderer}{Canvas\+Widget\+Renderer} to \doxylink{classtouchgfx_1_1_drawable_a602c55c069e516153167c3ebc7da4c0d}{move\+To()} the first coordinate and then line\+To() the next coordinates in order. Finally it tells CWR to render the inside of the shape using the set Painter object.

The \doxylink{classtouchgfx_1_1_line}{Line} class caches the four corners of the shape to speed up redrawing. In general, drawing lines involve some extra mathematics for calculating the normal vector of the line and this computation would slow down re-\/draws if not cached.

\begin{DoxyNote}{Note}
