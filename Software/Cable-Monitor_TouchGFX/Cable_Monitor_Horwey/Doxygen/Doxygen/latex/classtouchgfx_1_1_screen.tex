\doxysubsection{Detailed Description}
A \doxylink{classtouchgfx_1_1_screen}{Screen} represents a full-\/screen drawable area. Applications create specific screens by subclassing this class. Each \doxylink{classtouchgfx_1_1_screen}{Screen} has a root container to which drawables are added. The \doxylink{classtouchgfx_1_1_screen}{Screen} makes sure to delegate draw requests and various events to the appropriate drawables in correct order.
