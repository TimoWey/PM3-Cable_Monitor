\doxysubsection{Detailed Description}
Simple widget capable of drawing a circle, or part of a circle (an arc). The \doxylink{classtouchgfx_1_1_circle}{Circle} can be filled or be drawn as a simple line along the circumference of the circle. Several parameters of the circle can be changed\+: Center, radius, line width, line cap, start angle and end angle.

\begin{DoxyNote}{Note}
Since the underlying Canwas\+Widget\+Renderer only supports straight lines, the circle is drawn using many small straight lines segments. The granularity can be adjusted to match the requirements -\/ large circles need more line segments, small circles need fewer line segments, to look smooth and round. 

All circle parameters are internally handled as \doxylink{classtouchgfx_1_1_c_w_r_util_1_1_q5}{CWRUtil\+::\+Q5} which means that floating point values are rounded down to a fixed number of binary digits, for example\+:

Sets precision of the \doxylink{classtouchgfx_1_1_circle}{Circle} drawing function. The number given as precision is the number of degrees used as step counter when drawing the line fragments around the circumference of the circle, five being a reasonable value. Higher values results in less nice circles but faster rendering and possibly sufficient for very small circles. Large circles might require a precision smaller than five to make the edge of the circle look nice and smooth.
