\subsubsection*{template$<$class T$>$\newline
class touchgfx\+::\+Box\+With\+Border\+Button\+Style$<$ T $>$}
A box with border button style. This class is supposed to be used with one of the Button\+Trigger classes to create a functional button. This class will show a box with a border in different colors depending on the state of the button (pressed or released).

An image button style. This class is supposed to be used with one of the Button\+Trigger classes to create a functional button. This class will show one of two images depending on the state of the button (pressed or released).
