\doxysubsection{Detailed Description}
\doxylink{classtouchgfx_1_1_slide_menu}{Slide\+Menu} is a menu that can expand and collapse at the touch of a button. The \doxylink{classtouchgfx_1_1_slide_menu}{Slide\+Menu} can expand in any of the four directions. Menu items can be added, just like items are added to a normal container.

The relative positions of the background and state change button is configurable as is the direction in which the \doxylink{classtouchgfx_1_1_slide_menu}{Slide\+Menu} expands and collapses. How much of the \doxylink{classtouchgfx_1_1_slide_menu}{Slide\+Menu} that is visible when collapsed can also be set with the. It is, of course, important that the state change button is accessible when collapsed. The \doxylink{classtouchgfx_1_1_slide_menu}{Slide\+Menu} will collapse after a given timeout is reached. The timer can be reset, for example when the user interacts with elements in the list.

Menu elements are added normally using the \doxylink{classtouchgfx_1_1_slide_menu_adac2bd3bd618821854ca1fa631f03624}{add()} method and are positioned relative to the \doxylink{classtouchgfx_1_1_slide_menu}{Slide\+Menu}.
