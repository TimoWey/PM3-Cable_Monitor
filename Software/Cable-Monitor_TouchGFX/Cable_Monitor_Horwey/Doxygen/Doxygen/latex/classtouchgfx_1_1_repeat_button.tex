\doxysubsection{Detailed Description}
A \doxylink{classtouchgfx_1_1_repeat_button}{Repeat\+Button} is similar to a regular \doxylink{classtouchgfx_1_1_button}{Button}, but it will \textquotesingle{}repeat\textquotesingle{} if pressed for a long period of time. The \doxylink{classtouchgfx_1_1_repeat_button}{Repeat\+Button} differs from a regular \doxylink{classtouchgfx_1_1_button}{Button} with regards to activation. A \doxylink{classtouchgfx_1_1_button}{Button} is activated when the button is released, whereas a \doxylink{classtouchgfx_1_1_repeat_button}{Repeat\+Button} is activated immediately when pressed and then at regular intervals. A \doxylink{classtouchgfx_1_1_repeat_button}{Repeat\+Button} does not activate when released.

As for other well-\/known repeat buttons, the interval from the first activation until the second activation as well as the subsequent interval between activations can be set for the \doxylink{classtouchgfx_1_1_repeat_button}{Repeat\+Button}.

The default values for initial delay is 10 ticks, and the default value for the following delays between button activations is 5 ticks.
