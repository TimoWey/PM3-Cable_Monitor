\doxysubsection{Detailed Description}
This widget is capable of showing a text area on the screen. The text must be a predefined \doxylink{classtouchgfx_1_1_typed_text}{Typed\+Text} in the text sheet in the assets folder. This can be very useful when a font is an italic font where letters such as "{}j"{} and "{}g"{} extend a lot to the left under the previous character(s). if a line starts with a "{}j"{} or "{}g"{} this letter would either have to be pushed to the right to be able to see all of it, e.\+g. using spaces (which would ruin a multi line text which is left aligned) -\/ or by clipping the first letter (which could ruin the nice graphics). \begin{DoxyCode}{0}

\end{DoxyCode}
 Characters that do not extend to the left under the previous characters will be drawn in the same position in either case, but "{}j"{} and "{}g"{} will be aligned with other lines.
