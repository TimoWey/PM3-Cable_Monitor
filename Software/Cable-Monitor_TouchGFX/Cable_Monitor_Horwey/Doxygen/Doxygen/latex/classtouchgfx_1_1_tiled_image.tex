\doxysubsection{Detailed Description}
Simple widget capable of showing a bitmap tiled indefinitely horizontally and vertically. This means that when the \doxylink{classtouchgfx_1_1_tiled_image}{Tiled\+Image} \doxylink{classtouchgfx_1_1_widget}{Widget} is larger than the provided \doxylink{classtouchgfx_1_1_bitmap}{Bitmap}, the \doxylink{classtouchgfx_1_1_bitmap}{Bitmap} is repeated over and over horizontally and vertically. The bitmap can be alpha-\/ blended with the background and have areas of transparency.

Get (the largest possible) rectangle that is guaranteed to be solid (opaque). This information is important, as any \doxylink{classtouchgfx_1_1_drawable}{Drawable} underneath the solid area does not need to be drawn.

\end{DoxyReturn}
\begin{DoxyNote}{Note}

If no area can be guaranteed to be solid, an empty \doxylink{classtouchgfx_1_1_rect}{Rect} must be returned. Failing to return the correct rectangle may result in errors on the display. 
\end{DoxyNote}
