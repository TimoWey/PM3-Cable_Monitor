\doxysubsection{Detailed Description}
Radio button with two states. A \doxylink{classtouchgfx_1_1_radio_button}{Radio\+Button} is a button that changes appearance (state) when it has been pushed. Pushing the \doxylink{classtouchgfx_1_1_radio_button}{Radio\+Button} again will return the to original state.

To make managing radio buttons much easier, they can be added to a \doxylink{classtouchgfx_1_1_radio_button_group}{Radio\+Button\+Group} which then automates deselecting radio buttons when a new radio button is pressed.

Sets whether or not it is possible to deselect the \doxylink{classtouchgfx_1_1_radio_button}{Radio\+Button} by clicking it. By default it is not possible to deselect a \doxylink{classtouchgfx_1_1_radio_button}{Radio\+Button}. The meaning of this is most clear when the \doxylink{classtouchgfx_1_1_radio_button}{Radio\+Button} is used in a \doxylink{classtouchgfx_1_1_radio_button_group}{Radio\+Button\+Group} where exactly one \doxylink{classtouchgfx_1_1_radio_button}{Radio\+Button} should always be selected. Pressing the currently selected \doxylink{classtouchgfx_1_1_radio_button}{Radio\+Button} should not deselect it, but rather select it again. This makes the button "{}sticky"{}, i.\+e. a button can only be deselected by selecting another \doxylink{classtouchgfx_1_1_radio_button}{Radio\+Button} in the same \doxylink{classtouchgfx_1_1_radio_button_group}{Radio\+Button\+Group}.
