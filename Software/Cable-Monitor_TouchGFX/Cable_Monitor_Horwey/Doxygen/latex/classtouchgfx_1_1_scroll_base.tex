\doxysubsection{Detailed Description}
The \doxylink{classtouchgfx_1_1_scroll_base}{Scroll\+Base} class is an abstract class used for Widgets that needs to show (a lot of) elements in a \doxylink{classtouchgfx_1_1_drawable_list}{Drawable\+List} that can be scrolled. Due to memory limitations, this is implemented by re-\/using the Drawables in the \doxylink{classtouchgfx_1_1_drawable_list}{Drawable\+List} -\/ once an element is moved off screen, it is filled with new content and moved to the other end and the of the scrolling list.

Lists can be horizontal or vertical and the can be circular (infinite scrolling).
