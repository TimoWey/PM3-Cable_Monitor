\doxysubsection{Detailed Description}
The \doxylink{classtouchgfx_1_1_transition}{Transition} class is the base class for Transitions. Implementations of \doxylink{classtouchgfx_1_1_transition}{Transition} defines what happens when transitioning between Screens, which typically involves visual effects. An example of a transition implementation can be seen in example custom\+\_\+transition\+\_\+example. The most basic transition is the \doxylink{classtouchgfx_1_1_no_transition}{No\+Transition} class that does a transition without any visual effects.
