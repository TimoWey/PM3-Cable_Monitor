class touchgfx\+::\+Partition$<$ List\+Of\+Types, NUMBER\+\_\+\+OF\+\_\+\+ELEMENTS $>$}
This type provides a concrete \doxylink{classtouchgfx_1_1_partition}{Partition} of memory-\/slots capable of holding any of the specified list of types.

The \doxylink{classtouchgfx_1_1_partition}{Partition} is not aware of the types stored in the \doxylink{classtouchgfx_1_1_partition}{Partition} memory, hence it provides no mechanism for deleting C++ objects when the \doxylink{classtouchgfx_1_1_partition}{Partition} is \doxylink{classtouchgfx_1_1_abstract_partition_a2d494482f112b4fd326cf7aecc643626}{clear()}\textquotesingle{}ed.

This class implements \doxylink{classtouchgfx_1_1_abstract_partition}{Abstract\+Partition}.
