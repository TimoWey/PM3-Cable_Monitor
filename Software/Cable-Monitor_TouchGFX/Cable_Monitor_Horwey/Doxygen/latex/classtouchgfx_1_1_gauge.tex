\doxysubsection{Detailed Description}
A gauge is a graphical element that shows a needle on a dial, often a speedometer or similar. Much like a progress indicator, the minimum and maximum value of the \doxylink{classtouchgfx_1_1_gauge}{Gauge}, as well as steps can be set. For more information on this, consult the documentation on Progress\+Indicators.

A \doxylink{classtouchgfx_1_1_gauge}{Gauge} has a needle and optionally an arc that follows the needle.
