\doxysubsection{Detailed Description}
Class for optimizing and wrapping move and zoom operations on a \doxylink{classtouchgfx_1_1_scalable_image}{Scalable\+Image}. The \doxylink{classtouchgfx_1_1_zoom_animation_image}{Zoom\+Animation\+Image} takes two bitmaps representing the same image but at a small and a large resolution. These bitmaps should be the sizes that are used when not animating the image. The \doxylink{classtouchgfx_1_1_zoom_animation_image}{Zoom\+Animation\+Image} will use an \doxylink{classtouchgfx_1_1_image}{Image} for displaying the \doxylink{classtouchgfx_1_1_bitmap}{Bitmap} when its width and height matches either of them. When it does not match the size of one of the bitmaps, it will use a \doxylink{classtouchgfx_1_1_scalable_image}{Scalable\+Image} instead. The main idea is that the supplied bitmaps should be the end points of the zoom animation so that it ends up using an \doxylink{classtouchgfx_1_1_image}{Image} when not animating. This is, however, not a requirement. You can animate from and to sizes that are not equal the sizes of the bitmaps. The result is a container that has the high performance of an ordinary image when the size matches the pre-\/rendered bitmaps. Moreover it supplies easy to use animation functions that lets you zoom and move the image.

\begin{DoxyNote}{Note}
Since this container uses the \doxylink{classtouchgfx_1_1_scalable_image}{Scalable\+Image} it has the same restrictions as a Scaleable\+Image, i.\+e. 1bpp is not supported. 
\end{DoxyNote}
