\doxysubsection{Detailed Description}
The \doxylink{classtouchgfx_1_1_application}{Application} class is the main interface for manipulating screen contents. It holds a pointer to the currently displayed \doxylink{classtouchgfx_1_1_screen}{Screen}, and delegates draw requests and events to that \doxylink{classtouchgfx_1_1_screen}{Screen}. Additionally it contains some global application settings.
