\doxysubsection{Detailed Description}
A \doxylink{classtouchgfx_1_1_button}{Button} that has a bitmap with an icon on top of it. It is possible to have two different icons depending on the current state of the button (released or pressed).

Typical use case could be a blue button with a released and a pressed image. Those images can be reused across several buttons. The \doxylink{classtouchgfx_1_1_button_with_icon}{Button\+With\+Icon} will then allow an image to superimposed on top of the blue button.

Sets the four bitmaps used by this button. The last two bitmaps are drawn on top of the first two, again depending on the current state of the \doxylink{classtouchgfx_1_1_button}{Button}. This means that when the button state is released (normal), the new\+Icon\+Released is drawn on top of the new\+Background\+Released, and when the button state is pressed, the new\+Icon\+Pressed is drawn on top of the new\+Background\+Pressed.

The default position if the icons is set to the center of the bitmaps. The two icons are assumed to have the same dimensions. The size of the released icon is used to position the icons centered on the \doxylink{classtouchgfx_1_1_button}{Button}.
