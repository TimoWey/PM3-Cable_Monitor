\doxysubsection{Detailed Description}
A widget for displaying a buffer of pixel data. This can also be though of as a dynamic bitmap where the dimensions of the bitmap is the same as the dimensions of the widget and the actual bitmap data can be set and updated dynamically. The size of the buffer must match the number of bytes required for the widget calculated as WIDTH x HEIGHT x BYTES\+\_\+\+PER\+\_\+\+PIXEL. If the \doxylink{classtouchgfx_1_1_l_c_d}{LCD} is 16 bit per pixel the buffer must hold 2 bytes for each pixel. If the \doxylink{classtouchgfx_1_1_l_c_d}{LCD} is 24 bit the buffer must hold 3 bytes for each pixel.
