\doxysubsection{Detailed Description}
Simple widget capable of showing a bitmap on the display. The bitmap can be alpha-\/blended with the background (or whichever other \doxylink{classtouchgfx_1_1_drawable}{Drawable} might be "{}underneath"{} the \doxylink{classtouchgfx_1_1_image}{Image}). The bitmap can and have areas of varying opacity.

The conversion from image.\+bmp or image.\+png to a bitmap that can be used in Touch\+GFX is handled by the \doxylink{classtouchgfx_1_1_image}{Image} Converter as part of compiling the project. Each image is assigned a unique BITMAP identifier which.

Get (the largest possible) rectangle that is guaranteed to be solid (opaque). This information is important, as any \doxylink{classtouchgfx_1_1_drawable}{Drawable} underneath the solid area does not need to be drawn.

\end{DoxyReturn}
\begin{DoxyNote}{Note}

\end{DoxyNote}
