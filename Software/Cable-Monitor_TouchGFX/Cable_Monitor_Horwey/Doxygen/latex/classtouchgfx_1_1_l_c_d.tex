\doxysubsection{Detailed Description}
This class contains the various low-\/level drawing routines for drawing bitmaps, texts and rectangles/boxes. Normally, these draw operations are called from widgets, which also keep track of logical states such as visibility etc.

The \doxylink{classtouchgfx_1_1_l_c_d}{LCD} class cannot be instantiated, instead use one of the subclasses which implements the \doxylink{classtouchgfx_1_1_l_c_d}{LCD} drawing operations for a specific display configuration.

\begin{DoxyNote}{Note}
All coordinates sent to functions in the \doxylink{classtouchgfx_1_1_l_c_d}{LCD} class are expected to be in absolute coordinates, i.\+e. (0, 0) is upper left corner of the display. 
\end{DoxyNote}
