\doxysubsection{Detailed Description}
An analog clock. Should be supplied with images for the background, hour hand, minute hand and the optional second hand. You setup the \doxylink{classtouchgfx_1_1_analog_clock}{Analog\+Clock} by specifying the rotation point of each hand as well as the global rotation point of the clock. You can customize the behavior of the \doxylink{classtouchgfx_1_1_analog_clock}{Analog\+Clock} in respect to animations and relations between the hands e.\+g. if the hour hand should move gradually towards the next hour as the minute hand progresses (\doxylink{classtouchgfx_1_1_analog_clock_a656d9c9fd7e0e5d880d5c3098c039389}{set\+Hour\+Hand\+Minute\+Correction()})
