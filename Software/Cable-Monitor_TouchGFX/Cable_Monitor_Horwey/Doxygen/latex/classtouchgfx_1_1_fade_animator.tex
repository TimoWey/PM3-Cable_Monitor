\doxysubsection{Detailed Description}
\subsubsection*{template$<$class T$>$\newline
class touchgfx\+::\+Fade\+Animator$<$ T $>$}
A \doxylink{classtouchgfx_1_1_fade_animator}{Fade\+Animator} makes the template class T able to animate the alpha value from its current value to a specified end value. It is possible to use a fade in effect as well as fade out effect using \doxylink{classtouchgfx_1_1_fade_animator}{Fade\+Animator}. The alpha progression can be described by supplying an Easing\+Equation. The \doxylink{classtouchgfx_1_1_fade_animator}{Fade\+Animator} performs a callback when the animation has finished.

This mixin can be used on any \doxylink{classtouchgfx_1_1_drawable}{Drawable} that has a \textquotesingle{}void set\+Alpha(uint8\+\_\+t)\textquotesingle{} and a \textquotesingle{}uint8\+\_\+t get\+Alpha()\textquotesingle{} method.


\begin{DoxyTemplParams}{Template Parameters}
