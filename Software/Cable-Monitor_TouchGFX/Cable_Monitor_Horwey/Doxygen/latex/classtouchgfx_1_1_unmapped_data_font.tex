\doxysubsection{Detailed Description}
An \doxylink{classtouchgfx_1_1_unmapped_data_font}{Unmapped\+Data\+Font} has both glyph table and glyph data placed in a flash which does not support random access read (indirect access). A unicode table is located in a flash with random read access (direct access).
