\doxysubsection{Detailed Description}
The \doxylink{classtouchgfx_1_1_box_with_border}{Box\+With\+Border} class is used to create objects that can draw a box with a border on the display. The width of the border can be specified. If the border width is 0 the \doxylink{classtouchgfx_1_1_box_with_border}{Box\+With\+Border} will function just like a \doxylink{classtouchgfx_1_1_box}{Box}.
