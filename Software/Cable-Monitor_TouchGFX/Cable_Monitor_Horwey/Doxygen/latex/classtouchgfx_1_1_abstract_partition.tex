\doxysubsection{Detailed Description}
This type defines an abstract interface to a storage partition for allocating memory slots of equal size. The "{}partition"{} is not aware of the actual types stored in the partition memory, hence it provides no mechanism for deleting C++ objects when \doxylink{classtouchgfx_1_1_abstract_partition_a2d494482f112b4fd326cf7aecc643626}{clear()}\textquotesingle{}ed.

Since the \doxylink{classtouchgfx_1_1_partition}{Partition} concept is loosely typed this method shall be used with care.
