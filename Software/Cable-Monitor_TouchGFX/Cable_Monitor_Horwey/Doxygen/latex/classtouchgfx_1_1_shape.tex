\subsubsection*{template$<$uint16\+\_\+t POINTS$>$\newline
class touchgfx\+::\+Shape$<$ POINTS $>$}
Simple widget capable of drawing a fully filled shape. The shape can be scaled and rotated. The \doxylink{classtouchgfx_1_1_shape}{Shape} class allows the user to draw any shape and allows the defined shape to be scaled, rotated and moved freely. Example uses could be the hands of a clock.
