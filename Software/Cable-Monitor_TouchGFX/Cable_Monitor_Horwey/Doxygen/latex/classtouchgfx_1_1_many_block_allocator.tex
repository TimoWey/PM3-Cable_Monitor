class touchgfx\+::\+Many\+Block\+Allocator$<$ block\+\_\+size, blocks, bytes\+\_\+pr\+\_\+pixel $>$}
This class is partial framebuffer allocator using multiple blocks. New buffers can be allocated until no free blocks are available. After transfer to \doxylink{classtouchgfx_1_1_l_c_d}{LCD}, a block is queued for allocation again.
