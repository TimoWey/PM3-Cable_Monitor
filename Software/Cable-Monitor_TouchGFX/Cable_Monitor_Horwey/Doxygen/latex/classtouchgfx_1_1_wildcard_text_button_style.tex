\doxysubsection{Detailed Description}
\subsubsection*{template$<$class T$>$\newline
class touchgfx\+::\+Wildcard\+Text\+Button\+Style$<$ T $>$}
A wildcard text button style.

An wildcard text button style. This class is supposed to be used with one of the Button\+Trigger classes to create a functional button. This class will show a text with a wildcard in one of two colors depending on the state of the button (pressed or released).

The \doxylink{classtouchgfx_1_1_wildcard_text_button_style}{Wildcard\+Text\+Button\+Style} does not set the size of the enclosing container (normally \doxylink{classtouchgfx_1_1_abstract_button_container}{Abstract\+Button\+Container}). The size must be set manually.

To get a background behind the text, use \doxylink{classtouchgfx_1_1_wildcard_text_button_style}{Wildcard\+Text\+Button\+Style} together with e.\+g. \doxylink{classtouchgfx_1_1_image_button_style}{Image\+Button\+Style}\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{WildcardTextButtonStyle<ImageButtonStyle<ClickButtonTrigger>\ >\ myButton;}

\end{DoxyCode}


The position of the text can be adjusted with set\+Text\+XY (default is centered).
