\doxysubsection{Detailed Description}
A specialization of the Touch\+GFX \doxylink{classtouchgfx_1_1_application}{Application} class that provides the necessary glue for transitioning between presenter/view pairs.

It maintains a callback for transitioning and evaluates this at each tick.
