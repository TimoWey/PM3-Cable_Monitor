\doxysubsection{Detailed Description}
A repeat button trigger. This trigger will create a button that reacts to a consistent touch. This means it will call the set action repeatedly as long as it is touched. The \doxylink{classtouchgfx_1_1_repeat_button_trigger}{Repeat\+Button\+Trigger} can be combined with one or more of the Button\+Style classes to create a fully functional button.
