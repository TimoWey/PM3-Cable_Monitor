\doxysubsection{Detailed Description}
A scroll wheel is very much like the digit selector on a padlock with numbers. The digits always snap into place and exactly one number is always the "{}selected"{} number. Similar to an ordinary \doxylink{classtouchgfx_1_1_scroll_wheel}{Scroll\+Wheel}, but with a different style for the selected item which can thus be bold, have a different color or similar effect to highlight it and help create a 3D effect using very few resources.
