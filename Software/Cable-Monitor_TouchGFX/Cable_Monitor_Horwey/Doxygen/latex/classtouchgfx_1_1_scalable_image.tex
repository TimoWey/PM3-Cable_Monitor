\doxysubsection{Detailed Description}
\doxylink{classtouchgfx_1_1_widget}{Widget} for representing a scaled version of a bitmap. Simply change the width/height of the widget to resize the image. The quality of the scaled image depends of the rendering algorithm used. The rendering algorithm can be changed dynamically. Please note that scaling images is done at runtime and may require a lot of calculations.

\begin{DoxyNote}{Note}
Note that this widget does not support 1 bit per pixel color depth. 
\end{DoxyNote}

{\footnotesize\ttfamily \mbox{\hyperlink{classtouchgfx_1_1_rect}{Rect}} touchgfx\+::\+Scalable\+Image\+::get\+Solid\+Rect (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption}) const\hspace{0.3cm}{\ttfamily [virtual]}}

Get (the largest possible) rectangle that is guaranteed to be solid (opaque). This information is important, as any \doxylink{classtouchgfx_1_1_drawable}{Drawable} underneath the solid area does not need to be drawn.

\begin{DoxyReturn}{Returns}
The solid rectangle part of the \doxylink{classtouchgfx_1_1_drawable}{Drawable}.
\end{DoxyReturn}
\begin{DoxyNote}{Note}
The rectangle returned must be relative to upper left corner of the \doxylink{classtouchgfx_1_1_drawable}{Drawable}, meaning that a completely solid widget should return the full size \doxylink{classtouchgfx_1_1_rect}{Rect}(0, 0, \doxylink{classtouchgfx_1_1_drawable_a3a644571efd15a4809acd2717eff7b48}{get\+Width()}, \doxylink{classtouchgfx_1_1_drawable_a7edff68e4197ec124a1a6b7a446b4ede}{get\+Height()}). If no area can be guaranteed to be solid, an empty \doxylink{classtouchgfx_1_1_rect}{Rect} must be returned. Failing to return the correct rectangle may result in errors on the display. 
\end{DoxyNote}
