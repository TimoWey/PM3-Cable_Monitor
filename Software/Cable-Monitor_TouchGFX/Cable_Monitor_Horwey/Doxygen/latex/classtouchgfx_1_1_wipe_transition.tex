\subsubsection*{template$<$\mbox{\hyperlink{namespacetouchgfx_ae14b2b38c7bc9524a48095c970813102}{Direction}} template\+Direction$>$\newline
class touchgfx\+::\+Wipe\+Transition$<$ template\+Direction $>$}
A \doxylink{classtouchgfx_1_1_transition}{Transition} that expands the new screen over the previous from the given direction. This transition only draws the pixels in the framebuffer once, and never moves any pixels. It is therefore very useful on MCUs with limited performance.
