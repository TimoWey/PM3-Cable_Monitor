\subsubsection*{template$<$class T$>$\newline
class touchgfx\+::\+Snapper$<$ T $>$}
A mix-\/in that will make class T draggable and able to snap to a position when a drag operation has ended. The mix-\/in is able to perform callbacks when the snapper gets dragged and when the \doxylink{classtouchgfx_1_1_snapper}{Snapper} snaps to its snap position.
