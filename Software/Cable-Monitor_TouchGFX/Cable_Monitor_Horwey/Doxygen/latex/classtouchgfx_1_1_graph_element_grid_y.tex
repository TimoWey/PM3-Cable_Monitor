\doxysubsection{Detailed Description}
\doxylink{classtouchgfx_1_1_graph_element_grid_y}{Graph\+Element\+GridY} draws horizontal lines at selected intervals along the y axis. By combining two \doxylink{classtouchgfx_1_1_graph_element_grid_y}{Graph\+Element\+GridY} instances, it is possible to have minor and major grid lines.

\begin{DoxyNote}{Note}
The grid lines are drawn using \doxylink{classtouchgfx_1_1_l_c_d_ad91b3d6a891d588cc22fb5d25f1279cb}{LCD\+::fill\+Rect} for higher performance. 
\end{DoxyNote}
