\doxysubsection{Detailed Description}
A drawable list items interface. Used to pass the allocated array of drawable elements to \doxylink{classtouchgfx_1_1_scroll_list_a096661aa8c23cb951f78a2afd5d27e85}{Scroll\+List\+::set\+Drawables()}, \doxylink{classtouchgfx_1_1_scroll_wheel_ac6adf7ef0bf8ec5dcd7faa251bc97ea7}{Scroll\+Wheel\+::set\+Drawables()} or \doxylink{classtouchgfx_1_1_scroll_wheel_with_selection_style_a472c39359300751df50331ee4606c3d3}{Scroll\+Wheel\+With\+Selection\+Style\+::set\+Drawables()}. Provides easy access to each element in the array as well as the size of the array.
