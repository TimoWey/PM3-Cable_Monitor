\doxysubsection{Detailed Description}
\subsubsection*{template$<$class T$>$\newline
class touchgfx\+::\+Move\+Animator$<$ T $>$}
A \doxylink{classtouchgfx_1_1_move_animator}{Move\+Animator} makes the template class T able to animate a movement from its current position to a specified end position. The speed of the movement in both the X and Y direction can be controlled by supplying \doxylink{classtouchgfx_1_1_easing_equations}{Easing\+Equations}. The \doxylink{classtouchgfx_1_1_move_animator}{Move\+Animator} performs a callback when the animation has finished.

This mixin can be used on any \doxylink{classtouchgfx_1_1_drawable}{Drawable}.
