\doxysubsection{Detailed Description}
The \doxylink{classtouchgfx_1_1_drawable}{Drawable} class is an abstract definition of something that can be drawn. In the composite design pattern, the \doxylink{classtouchgfx_1_1_drawable}{Drawable} is the component interface. Drawables can be added to a screen as a tree structure through the leaf node class \doxylink{classtouchgfx_1_1_widget}{Widget} and the \doxylink{classtouchgfx_1_1_container}{Container} class. A \doxylink{classtouchgfx_1_1_drawable}{Drawable} contains a pointer to its next sibling and a pointer to its parent node. These are maintained by the \doxylink{classtouchgfx_1_1_container}{Container} to which the \doxylink{classtouchgfx_1_1_drawable}{Drawable} is added.

The \doxylink{classtouchgfx_1_1_drawable}{Drawable} interface contains two pure virtual functions which must be implemented by widgets, namely \doxylink{classtouchgfx_1_1_drawable_a9ad1411de4b9e2c6c6abb107d349c021}{draw()} and \doxylink{classtouchgfx_1_1_drawable_ab699c8781fc81aed49c46ff19b30b0dd}{get\+Solid\+Rect()}. In addition it contains general functionality for receiving events and navigating the tree structure.

The coordinates of a \doxylink{classtouchgfx_1_1_drawable}{Drawable} are always relative to its parent node.
