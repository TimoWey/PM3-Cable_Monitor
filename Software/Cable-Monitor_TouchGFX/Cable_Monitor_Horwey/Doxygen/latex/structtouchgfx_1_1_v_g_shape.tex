\doxysubsection{Detailed Description}
Data structure for a vector shape (polygon, circle, path, etc.).

Shapes can be linked in a list using the next member. Shapes are linked to by a \doxylink{structtouchgfx_1_1_v_g_object}{VGObject}. Which thereby consists of a number of Shapes. Each shape has attributes, e.\+g. stroke\+Width, that is used when drawing the paths in the \doxylink{classtouchgfx_1_1_shape}{Shape}.

All coordinates and sizes (e.\+g. stroke\+Width) are not translated by any transform given in the svg. This has to be done when drawing.
