\doxysubsection{Detailed Description}
Defines the "{}\+Penner easing functions"{}, which are a de facto standard for computing aesthetically pleasing motion animations. See \href{http://easings.net/}{\texttt{ http\+://easings.\+net/}} for visual illustrations of the easing equations.
