\doxysubsection{Detailed Description}
Using \doxylink{classtouchgfx_1_1_line}{Line} from \doxylink{classtouchgfx_1_1_canvas_widget_renderer}{Canvas\+Widget\+Renderer}, progress will be rendered as a line. This means that the user must create a painter for painting the circle. The line does not need to horizontal or vertical, but can start at any coordinate and finish at any coordinate.

\begin{DoxyNote}{Note}
As \doxylink{classtouchgfx_1_1_line_progress}{Line\+Progress} uses \doxylink{classtouchgfx_1_1_canvas_widget_renderer}{Canvas\+Widget\+Renderer}, it is important that a buffer is set up by calling Canvas\+Widget\+Rendere\+::set\+Buffer(). 
\end{DoxyNote}
