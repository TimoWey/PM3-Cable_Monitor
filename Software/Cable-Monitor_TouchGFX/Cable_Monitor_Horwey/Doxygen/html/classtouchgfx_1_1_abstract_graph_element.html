Detailed Description

An abstract graph element. Declares a couple of useful functions to help subclasses which use CWR (Canvas Widget Renderer).

◆ invalidateGraphPointAt()

virtual void touchgfx::AbstractGraphElement::invalidateGraphPointAt ( int16_t index)
pure virtual

Invalidate the point at the given index. This allows a graph element to only invalidate the minimum rectangle required for the given index. The Graph will call this function before and after changing a point to ensure that both the old and the new area are redrawn (invalidated).

Parameters
indexZero-based index of the point.

Implemented in touchgfx::GraphElementArea, touchgfx::GraphElementLine, touchgfx::GraphElementVerticalGapLine, touchgfx::GraphElementHistogram, touchgfx::GraphElementBoxes, touchgfx::GraphElementDots, touchgfx::GraphElementDiamonds, touchgfx::GraphLabelsX, touchgfx::GraphElementGridBase, touchgfx::GraphLabelsBase, and touchgfx::GraphTitle.
