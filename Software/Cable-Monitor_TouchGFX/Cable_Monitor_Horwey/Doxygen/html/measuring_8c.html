Detailed Description

@brief Measuring input signal using following configuration:

  • ADC triggered by a timer and with interrupt after end of conversion
  • ADC combined with DMA (Direct Memory Access) to fill a buffer
  • Scan mode = sequential sampling of two inputs by one ADC
  • Analog mode configuration for GPIOs

Peripherals @ref HowTo

This Software is to be used with the STM32F4-Discovery Board in conjunction with the Cable-Monitor Board designed by A. Horvat and T. Wey.
The Cable-Monitor Board is connected to the STM32F4-Discovery Board

The following peripherals are used:

  • ADC3
  • DMA2
  • TIM2
  • GPIO
  • NVIC
  • RCC

How to Configure the Peripherals: ADC, TIMER and DMA

All the peripherals are accessed by writing to or reading from registers. From the programmer’s point of view this is done exactly as writing or reading the value of a variable.
Writing to a register configures the HW of the associated peripheral to do what is required.
Reading from a registers gets status and data from the HW peripheral.

The information on which bits have to be set to get a specific behavior is documented in the reference manual of the microcontroller.

◆ MEAS_ADC3_scan_init()

void MEAS_ADC3_scan_init ( void  )

Initialize ADC, timer and DMA for sequential acquisition = scan mode.

Uses ADC2 and DMA2_Stream1 channel2
The ADC2 trigger is set to TIM2 TRGO event
At each trigger both inputs are converted sequentially and transfered to memory by the DMA.
As each conversion triggers the DMA, the number of transfers is doubled.
The DMA triggers the transfer complete interrupt when all data is ready.
The inputs used are ADC3_IN4 = GPIO PF6 (Pad Right), ADC3_IN13 = GPIO PC3 (Pad Left), ADC_IN6 = GPIO PF8 (HS-Right), ADC3_IN11 = GPIO PC1 (HS-Left)

void MEAS_ADC_reset ( void  )

Resets the ADCs, disables the timer.

This function resets the ADCs by asserting the ADC reset bit in the RCC_APB2RSTR register, then releases the reset by clearing the ADC reset bit. It also disables the timer by clearing the TIM_CR1_CEN bit in the TIM2 control register.
