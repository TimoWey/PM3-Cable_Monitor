Detailed Description

Interface class for manipulating GPIOs in order to do performance measurements on target. Not used on the PC simulator.

Enum for the GPIOs used.

Enumerator
RENDER_TIME 

Pin is toggled at each VSYNC.

FRAME_RATE 

Pin is high when frame rendering begins, low when finished.

MCU_ACTIVE 

Pin is toggled when the framebuffers are swapped.

Pin is high when the MCU is doing work (i.e. not in idle task).
