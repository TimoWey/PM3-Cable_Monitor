   Checks whether the touch screen is being touched, and if so, what coordinates.
Parameters
[out]xThe x position of the touch
[out]yThe y position of the touch
Returns
True if a touch has been detected, otherwise false.

By default sampleTouch returns false, return true if a touch has been detected, otherwise false.

Coordinates are passed to the caller by reference by x and y.

This function is called by the TouchGFX framework. By default sampleTouch is called every tick, this can be adjusted by HAL::setTouchSampleRate(int8_t);
