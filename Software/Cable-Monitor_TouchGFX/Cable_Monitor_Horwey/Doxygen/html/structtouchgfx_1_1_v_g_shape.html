Detailed Description

Data structure for a vector shape (polygon, circle, path, etc.).

Shapes can be linked in a list using the next member. Shapes are linked to by a VGObject. Which thereby consists of a number of Shapes. Each shape has attributes, e.g. strokeWidth, that is used when drawing the paths in the Shape.

All coordinates and sizes (e.g. strokeWidth) are not translated by any transform given in the svg. This has to be done when drawing.
